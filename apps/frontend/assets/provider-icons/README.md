# Provider 图标资源

请在此目录放置以下 Provider 的图标文件：

## 所需图标文件

| 文件名 | Provider | 推荐尺寸 | 说明 |
|--------|----------|----------|------|
| `openai.png` | OpenAI | 64x64 或 128x128 | OpenAI Logo（绿色或黑色） |
| `anthropic.png` | Anthropic | 64x64 或 128x128 | Anthropic Logo（紫色） |
| `azure_openai.png` | Azure OpenAI | 64x64 或 128x128 | Azure OpenAI Logo（蓝色） |
| `aliyun_codeplan.png` | 阿里云 Code Plan | 64x64 或 128x128 | 阿里云 Logo（橙色） |

## 图标获取建议

### OpenAI
- 官方 Logo: https://cdn.simpleicons.org/openai
- 或从 OpenAI 官网获取

### Anthropic
- 官方 Logo: https://cdn.simpleicons.org/anthropic
- 或从 Anthropic 官网获取

### Azure OpenAI
- Microsoft Fluent UI Icons: https://aka.ms/fluentui-system-icons
- 或使用 Azure 云图标

### 阿里云
- 阿里云 Logo: https://cdn.simpleicons.org/alibabacloud
- 或从阿里云官网获取

## 图标要求

- **格式**: PNG（透明背景）或 SVG
- **尺寸**: 64x64 或 128x128（会自动缩放到 24x24 显示）
- **背景**: 透明背景为佳
- **颜色**: 使用各品牌的标准色

## 临时方案

如果暂时无法获取图标，代码会显示 Unicode 字符 ⚪ 作为占位符。
