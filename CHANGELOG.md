## [v1.0.0] - 2026-04-03

### 核心功能
- **笔记管理**：Markdown 编辑器、标签分类、笔记 CRUD
- **碎片化内容采集**：微信公众号、B 站、YouTube 多平台导入器框架
- **知识图谱**：Neo4j 存储，LLM 智能提取概念与关系，2D 力导向图可视化
- **知识图谱增量更新**：全量构建与增量更新支持
- **知识图谱节点关联**：节点可关联笔记
- **AI 助手**：流式聊天，多 LLM 支持（OpenAI / Anthropic / Azure）
- **LLM 配置管理**：多模型配置，测试连接
- **后台任务调度**：Redis-backed 任务队列
- **用户认证**：OAuth2 + JWT，双因素认证（TOTP）
- **用户会话管理**：多设备会话管理

### 前端
- **三栏布局**：现代化的三栏布局组件
- **内容详情页**：优化布局和 Markdown 预览
- **Markdown 预览**：语法高亮支持
- **标签系统**：标签颜色、管理界面、侧边栏展示
- **平台筛选**：按内容来源筛选卡片
- **底部导航**：笔记 | 碎片 | 图谱 | 导图 | 任务 | 设置

### 后端
- **FastAPI 架构**：高性能异步 API
- **SQLAlchemy ORM**：完整的数据模型
- **数据库自动初始化**：启动时自动建表 + 创建 admin
- **后端测试框架**：pytest 单元测试

### 部署
- **Docker Compose**：开发/生产双环境配置
- **生产环境部署脚本**：一键启动/停止/备份
- **Nginx 反向代理**：前后端代理配置

### 基础设施
- **PostgreSQL**：主数据库
- **Redis**：缓存 + 任务队列
- **Neo4j**：知识图谱存储
- **图标库**：统一使用 lucide-react-native
